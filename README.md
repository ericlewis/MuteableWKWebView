# MuteableWKWebView (SwiftPM)

Mute per WKWebView like Safari.

----
# How to use

Load this project with SwiftPM.

```Swift
import MuteableWKWebView
```

MuteableWKWebView append mute property into WKWebView.

Usage:
```Swift

// mute
webView.muted = true

// unmute
webView.muted = false
```
